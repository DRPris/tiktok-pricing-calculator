# 东南亚TikTok Shop费用标准汇总

## 数据来源
- TikTok Shop官方商家大学
- 更新时间：2026年1月
- 来源：https://seller.tiktokglobalshop.com/university/essay?knowledge_id=6173886299686658

## 各国费率对比表

### 菲律宾 (Philippines)
- **平台佣金**：5.0% - 9.1% (含税，按子类目)
- **交易手续费**：2.24% (含税)
- **电商增长服务费**：暂无
- **平台基础设施费**：订单处理费 (Order Processing Fee)
- **VAT税率**：12%
- **关税**：0-30% (根据商品类目)
- **生效日期**：2025年1月15日

### 马来西亚 (Malaysia)
- **平台佣金**：按子类目收取，非BXP商家+4% (2025年8月15日生效)
- **交易手续费**：3.78% (含税)
- **电商增长服务费**：暂无
- **平台基础设施费**：暂无
- **SST税率**：8% (销售与服务税)
- **关税**：0-30% (根据商品类目)
- **新商家政策**：入驻起90天免佣

### 新加坡 (Singapore)
- **平台佣金**：3.27% (含税，固定佣金，2025年7月15日生效)
- **交易手续费**：2.18% (含税)
- **电商增长服务费**：暂无
- **平台基础设施费**：暂无
- **GST税率**：9% (商品及服务税)
- **关税**：大部分商品免税
- **低价值商品税**：2024年1月1日起征收

### 泰国 (Thailand)
- **平台佣金**：
  - 电子产品：5.35% (含税)
  - 其他品类：6.42% (含税)
- **交易手续费**：3.21% (含税，2024年4月生效)
- **电商增长服务费**：
  - 电子产品：5.35% (上限199泰铢/件)
  - 其他品类：6.42% (上限199泰铢/件)
- **平台基础设施费**：1.07泰铢/订单
- **VAT税率**：7%
- **关税**：0-30% (根据商品类目)
- **新商家政策**：前30天电商增长服务费免收(上限5000泰铢)

### 越南 (Vietnam)
- **平台佣金**：5% (含税，固定佣金，2025年7月15日生效)
- **交易手续费**：3.21% (含税)
- **电商增长服务费**：暂无
- **订单处理手续费**：存在
- **VAT税率**：10% (2025年2月18日起调整)
- **关税**：2025年2月23日起代扣高价值商品关税

## 通用规则

### 佣金计算公式
```
平台佣金 = 佣金率% × 商家折扣后价
```
- 运费和平台折扣不计入佣金基数

### 交易手续费计算公式
```
交易手续费 = (买家支付费用 - 买家退款) × 交易手续费率
```

### 新商家免佣政策
- 入驻后30天内完成任务，可获最长90天免佣期
- 马来西亚：入驻起90天直接免佣，无需完成任务
- 具体规则以后台任务展示为准

## 费用对比分析

### 总费率排名（从低到高）
1. **新加坡**：3.27% + 2.18% = 5.45%
2. **越南**：5% + 3.21% = 8.21%
3. **泰国**（电子产品）：5.35% + 3.21% + 5.35% = 13.91% (+ 1.07泰铢固定费)
4. **菲律宾**：5-9.1% + 2.24% = 7.24% - 11.34%
5. **马来西亚**：按类目 + 3.78% (+ 4%涨佣后更高)
6. **泰国**（其他品类）：6.42% + 3.21% + 6.42% = 16.05% (+ 1.07泰铢固定费)

### 税费对比
- **最低**：新加坡 (GST 9%，大部分商品免关税)
- **中等**：泰国 (VAT 7% + 关税0-30%)
- **较高**：菲律宾 (VAT 12% + 关税0-30%)、越南 (VAT 10% + 关税)
- **特殊**：马来西亚 (SST 8% + 关税0-30%)
